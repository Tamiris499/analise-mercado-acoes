# 📊 Análise Mercado de Ações

Projeto desenvolvido durante meus estudos em Análise de Dados.

## 🎯 Objetivo
Realizar análises de operações de compra e venda de ações, identificando tendências, estatísticas e comportamento dos clientes.

## 🛠️ Ferramentas utilizadas
- Google Planilhas
- Excel
- Análise de Dados
- Estatística

## 📈 Análises realizadas
- Clientes com prejuízo
- Tendência temporal
- Compra e venda de ações
- Estatísticas das operações
- Organização e tratamento de dados

## 📚 Aprendizados
Durante o desenvolvimento deste projeto, pratiquei:
- Limpeza e organização de dados
- Estruturação de tabelas
- Análise estatística
- Interpretação de informações
- Visualização de dados

## 🚀 Status do projeto
Projeto finalizado como prática de estudos em Análise de Dados.
